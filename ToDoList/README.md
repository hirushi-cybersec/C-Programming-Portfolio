
📝 C To-Do List Manager

A simple yet powerful command-line To-Do List application built in C.

---

✨ Features
- *➕ Add Tasks*: Add unlimited tasks to your list
- *👀 View Tasks*: Display all pending tasks with numbers
- *🗑️ Delete Tasks*: Remove completed tasks by number
- *💾 Memory Management*: Uses arrays for efficient storage
- *🖥️ CLI Interface*: Clean, user-friendly menu system

---

🛠️ How to Run
```bash
gcc todo.c -o todo
./todo
